cp Command

The cp command in Unix/Linux stands for "Copy". It is used to copy files and directories.

Syntax
cp [options] source_file(s) destination
Description

The cp command copies the source_file(s) to the destination. If the destination is an existing directory, the source_file(s) are copied into that directory.

Options
  • -r: Copy directories recursively.
  • -p: Preserve file attributes (timestamps, permissions).
  • -v: Verbose mode, output what is being done.
Examples
cp file1.txt file2.txt

This command copies file1.txt to file2.txt.

cp -r directory1 /path/to/directory2

This command recursively copies directory1 and its contents into /path/to/directory2.
