tr Command

The tr command in Unix/Linux translates or deletes characters. It reads characters from standard input, performs specified translations, and writes the result to standard output.

Syntax
tr [options] set1 set2
Description

The tr command translates characters from set1 to set2 according to specified rules. It can also delete characters specified in set1.

Options
  • -d: Delete characters in set1 from the input.
  • -c: Complement set1; replace characters not in set1.
  • -s: Squeeze multiple occurrences of characters in set2 to a single character.
Examples
echo "hello" | tr '[:lower:]' '[:upper:]'

This command converts lowercase letters in "hello" to uppercase.

echo "abc123xyz789" | tr -d '[:digit:]'

This command removes digits from the input string "abc123xyz789".
