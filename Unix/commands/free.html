free Command

The free command in Unix/Linux displays the amount of free and used memory in the system, including physical and swap memory.

Syntax
free [options]
Description

The free command shows the total amount of free and used physical and swap memory in the system, as well as the buffers used by the kernel.

Options

The free command typically does not have many options. Commonly used options include:

  • -h: Display memory sizes in human-readable format (e.g., MB, GB).
  • -s seconds: Update statistics every seconds.
Examples
free

This command displays memory usage statistics.

free -h

This command displays memory sizes in a human-readable format.

free -s 5

This command updates memory statistics every 5 seconds.
