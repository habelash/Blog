cat Command

The cat command in Unix/Linux concatenates and displays files. It is commonly used to display the contents of one or more files to standard output.

Syntax
cat [options] [file(s)]
Description

The cat command reads one or more files and copies them to standard output. It is also used to create, concatenate, and display files.

Options
  • -n: Number all output lines.
  • -b: Number non-blank output lines.
  • -s: Squeeze multiple adjacent blank lines into one.
Examples
cat file.txt

This command displays the contents of file.txt on the terminal.

cat file1.txt file2.txt > merged_file.txt

This command concatenates the contents of file1.txt and file2.txt and saves them to merged_file.txt.
