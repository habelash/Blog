mv Command

The mv command in Unix/Linux stands for "Move". It is used to move or rename files and directories.

Syntax
mv [options] source destination
Description

The mv command moves the source file or directory to the destination. If destination is an existing directory, source is moved into that directory.

Options
  • -i: Prompt before overwriting an existing file.
  • -v: Verbose mode, output what is being done.
  • --backup[=CONTROL]: Make a backup of each existing destination file.
Examples
mv file1.txt /path/to/directory/

This command moves file1.txt to /path/to/directory/.

mv file1.txt file2.txt

This command renames file1.txt to file2.txt.
