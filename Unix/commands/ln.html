ln Command

The ln command in Unix/Linux creates links between files. It can create hard links and symbolic (soft) links.

Syntax
ln [options] target link_name
Description

The ln command creates a link from link_name to target. By default, it creates hard links. Use the -s option to create symbolic (soft) links.

Options
  • -s: Create symbolic links instead of hard links.
  • -f: Force creation of links, overwrite existing files.
  • -v: Verbose mode, output what is being done.
Examples
ln file1.txt link1

This command creates a hard link named link1 pointing to file1.txt.

ln -s /path/to/file symlink

This command creates a symbolic link named symlink pointing to /path/to/file.
