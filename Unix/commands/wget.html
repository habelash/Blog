wget Command

The wget command in Unix/Linux retrieves files from the web using HTTP, HTTPS, or FTP protocols.

Syntax
wget [options] [URL]
Description

wget is a command-line utility for downloading files from the internet. It supports downloading files via HTTP, HTTPS, and FTP protocols, allowing users to specify options like saving the file under a different name, resuming interrupted downloads, and more.

Examples
wget https://example.com/file.zip

This command downloads a file from the specified URL.

wget -O output-file.txt https://example.com/data.txt

This command downloads a file and saves it as output-file.txt.

wget -c https://example.com/largefile.zip

This command resumes a partially downloaded file.
