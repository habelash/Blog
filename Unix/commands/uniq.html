uniq Command

The uniq command in Unix/Linux filters adjacent matching lines from input. It is used to remove or display duplicate lines in a file, considering adjacent lines or based on a specified count of occurrences.

Syntax
uniq [options] [input_file]
Description

The uniq command reads from standard input or a file and filters out adjacent matching lines. It can also count and display unique lines based on specified options.

Options
  • -c: Prefix lines by the number of occurrences.
  • -d: Only output duplicate lines, one per group.
  • -i: Ignore differences in case when comparing lines.
Examples
uniq file.txt

This command displays unique lines from file.txt.

uniq -c file.txt

This command counts occurrences of each unique line in file.txt.
