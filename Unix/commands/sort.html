sort Command

The sort command in Unix/Linux sorts lines of text files. It arranges lines alphabetically or numerically, and can also merge files together in a sorted order.

Syntax
sort [options] [file(s)]
Description

The sort command reads data from a file or standard input, sorts it line by line, and writes the result to standard output. It can sort in ascending or descending order, and handle numerical or textual data.

Options
  • -r: Reverse the result of comparisons.
  • -n: Sort numerically.
  • -k field: Sort based on a specific field or column.
Examples
sort file.txt

This command sorts lines in file.txt alphabetically.

sort -n numbers.txt

This command sorts lines in numbers.txt numerically.

sort -r -k 2 data.csv

This command sorts lines in data.csv in reverse order based on the second column.
