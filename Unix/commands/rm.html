rm Command

The rm command in Unix/Linux stands for "Remove". It is used to delete files and directories.

Syntax
rm [options] file(s) directory
Description

The rm command removes files or directories specified by file(s) or directory. Directories are removed recursively by default.

Options
  • -f: Forcefully remove without prompting for confirmation.
  • -r: Recursively remove directories and their contents.
  • -v: Verbose mode, output what is being done.
Examples
rm file1.txt

This command deletes file1.txt.

rm -r directory1

This command recursively deletes directory1 and its contents.
