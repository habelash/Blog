scp Command

The scp command in Unix/Linux securely copies files between hosts on a network.

Syntax
scp [options] [source] [destination]
Description

The scp command securely transfers files between hosts on a network using the SSH protocol. It can copy files between remote hosts, or from remote to local or local to remote.

Options
  • -r: Recursively copy entire directories.
  • -P port: Specify a different port for connection.
  • -i identity_file: Use a different identity file for authentication.
Examples
scp file.txt user@remote_host:/remote/directory/

This command copies file.txt from the local machine to /remote/directory/ on remote_host.

scp -r /local/directory/ user@remote_host:/remote/directory/

This command recursively copies /local/directory/ to /remote/directory/ on remote_host.

scp -P 2222 file.txt user@remote_host:/remote/directory/

This command copies file.txt using port 2222 to /remote/directory/ on remote_host.
