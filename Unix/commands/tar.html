tar Command

The tar command in Unix/Linux archives and extracts files.

Syntax
tar [options] [files or directories...]
Description

The tar command creates, extracts, or manipulates files in a compressed archive. It can handle various compression formats, including gzip and bzip2.

Options
  • -c: Create a new archive.
  • -x: Extract files from an archive.
  • -v: Verbose mode (show progress).
  • -f file: Use archive file or device file.
  • -z: Compress archive with gzip.
  • -j: Compress archive with bzip2.
  • -t: List the contents of an archive.
Examples
tar -cvf archive.tar file1 file2

This command creates a new tar archive archive.tar containing file1 and file2.

tar -xvf archive.tar

This command extracts the contents of archive.tar into the current directory.

tar -tvf archive.tar

This command lists the contents of archive.tar without extracting it.
