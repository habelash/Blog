grep Command

The grep command in Unix/Linux searches for lines matching a specified pattern in files or input streams.

Syntax
grep [options] pattern [file(s)]
Description

The grep command searches for lines containing a match to the specified pattern. It can be used to search files or as part of a pipeline for text processing.

Options
  • -i: Ignore case distinctions.
  • -r: Recursively search subdirectories.
  • -n: Show line numbers along with the output.
Examples
grep 'pattern' file.txt

This command searches for occurrences of 'pattern' in file.txt.

grep -r 'keyword' directory/

This command recursively searches for 'keyword' in all files under directory/.

grep -n 'error' log.txt

This command searches for lines containing 'error' in log.txt and displays line numbers.
