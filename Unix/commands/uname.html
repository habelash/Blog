uname Command

The uname command in Unix/Linux prints system information such as the system name, kernel version, and more.

Syntax
uname [options]
Description

The uname command displays essential information about the operating system and its configuration. It can show the system name, kernel version, machine hardware name, and more.

Options
  • -s: Print the kernel name.
  • -n: Print the network node hostname.
  • -r: Print the kernel release.
  • -v: Print the kernel version.
  • -m: Print the machine hardware name.
Examples
uname -s

This command prints the kernel name.

uname -r

This command prints the kernel release version.

uname -m

This command prints the machine hardware name.
