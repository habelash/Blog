uptime Command

The uptime command in Unix/Linux displays how long the system has been running.

Syntax
uptime
Description

The uptime command shows the current time, how long the system has been running, the number of users currently logged on, and the system load averages for the past 1, 5, and 15 minutes.

Output

The output typically includes:

  • The current time.
  • How long the system has been running.
  • Number of users logged in.
  • Load averages (1, 5, and 15 minutes).
Examples
uptime

This command displays the uptime and other system information.
