Introduction to Python

Python is a versatile, high-level programming language known for its readability and ease of use. It's widely used for web development, data analysis, artificial intelligence, and more.

To check your Python version, use:

python --version

Or for Python 3:

python3 --version

Basic Python Syntax

Hello World

print("Hello, World!")

Variables and Data Types

x = 10  # Integer
y = 3.14  # Float
name = "Alice"  # String
is_valid = True  # Boolean

Comments

# This is a single-line comment
'''
This is a
multi-line comment
'''

Control Structures

Conditional Statements

if x > 10:
    print("x is greater than 10")
elif x == 10:
    print("x is 10")
else:
    print("x is less than 10")

Loops

# For loop
for i in range(5):
    print(i)

# While loop
count = 0
while count < 5:
    print(count)
    count += 1

Functions and Modules

Defining a Function

def greet(name):
    return f"Hello, {name}!"

Using a Function

print(greet("Alice"))

Importing Modules

import math
print(math.sqrt(16))

Object-Oriented Programming

Classes and Objects

class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age
    
    def bark(self):
        return "Woof!"

dog = Dog("Buddy", 4)
print(dog.bark())

Working with Files

Reading a File

with open('example.txt', 'r') as file:
    content = file.read()
    print(content)

Writing to a File

with open('example.txt', 'w') as file:
    file.write("Hello, file!")

Exception Handling

Try-Except Block

try:
    x = 1 / 0
except ZeroDivisionError:
    print("You can't divide by zero!")

Libraries and Packages

Installing Packages

pip install numpy

Using a Package

import numpy as np
array = np.array([1, 2, 3])
print(array)

Virtual Environments

Creating a Virtual Environment

python -m venv myenv

Activating the Virtual Environment

# On Windows
myenv\Scripts\activate

# On macOS/Linux
source myenv/bin/activate

Common Python Commands

Running a Python Script

python script.py

Checking Installed Packages

pip list
