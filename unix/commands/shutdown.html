shutdown Command

The shutdown command in Unix/Linux is used to shut down or restart the system.

Syntax
shutdown [options] [time] [message]
Options
  • -h - Halt the system (shut down)
  • -r - Restart the system
  • -c - Cancel a pending shutdown
  • -t seconds - Specify the delay in seconds before shutdown/restart
Examples
shutdown -h now
shutdown -r +10 "System will restart in 10 minutes for maintenance"
