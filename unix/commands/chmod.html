chmod Command

The chmod command in Unix/Linux stands for "change mode". It is used to change the permissions (mode) of files and directories.

Syntax
chmod [options] mode file(s)
Description

The chmod command changes the permissions of file(s) according to mode. Permissions specify who can read, write, and execute the file or directory.

Options
  • -R: Recursively change permissions for directories and their contents.
  • -v: Verbose mode, output what is being done.
  • -c: Report only when a change is made.
Examples
chmod u+x file.sh

This command adds execute permission for the owner of file.sh.

chmod -R 755 directory

This command recursively sets read, write, and execute permissions for the owner, and read and execute permissions for others on directory and its contents.
