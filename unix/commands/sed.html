sed Command

The sed command in Unix/Linux is a stream editor used to perform basic text transformations on an input stream (a file or input from a pipeline).

Syntax
sed [options] 'command(s)' [file(s)]
Description

The sed command processes text line by line and applies specified commands (such as search and replace) to each line or a range of lines from the input.

Examples
sed 's/old/new/g' file.txt

This command replaces all occurrences of 'old' with 'new' in file.txt.

sed -n '1,10p' data.txt

This command prints lines 1 to 10 from data.txt.

sed '/pattern/d' file.txt

This command deletes lines from file.txt that contain the specified pattern.
