less Command

The less command in Unix/Linux is a pager that allows viewing text files one screen at a time. It provides navigation and search functionalities within the file.

Syntax
less [options] [file(s)]
Description

The less command displays text files, allowing scrolling through content using arrow keys. It supports search operations, navigation to specific line numbers, and can handle large files efficiently.

Options
  • -N: Display line numbers.
  • -i: Ignore case when searching.
  • -M: Display more detailed prompt.
Examples
less file.txt

This command opens file.txt for viewing and navigation.

less -N -i data.log

This command opens data.log with line numbers displayed and case-insensitive search.

less +10 largefile.txt

This command opens largefile.txt and jumps to line 10.
