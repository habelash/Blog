chown Command

The chown command in Unix/Linux stands for "change owner". It is used to change the owner and group of files and directories.

Syntax
chown [options] owner[:group] file(s)
Description

The chown command changes the owner and group of file(s) to owner and optionally group. This command is useful for changing file ownership after copying or creating files.

Options
  • -R: Recursively change ownership for directories and their contents.
  • -v: Verbose mode, output what is being done.
  • --from=oldowner[:oldgroup]: Change ownership from specified oldowner and optionally oldgroup.
Examples
chown user1:group1 file.txt

This command changes the owner of file.txt to user1 and the group to group1.

chown -R user2 directory

This command recursively changes the owner of directory and its contents to user2.
