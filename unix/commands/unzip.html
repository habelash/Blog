unzip Command

The unzip command in Unix/Linux extracts files from a ZIP archive.

Syntax
unzip [options] file.zip
Description

The unzip command extracts files from a ZIP archive. It can handle both encrypted and non-encrypted ZIP files and provides various options for extracting files.

Options
  • -d directory: Extract files into the specified directory.
  • -q: Quiet mode (suppress output).
  • -o: Overwrite existing files without prompting.
  • -p: Extract files to standard output (pipe mode).
Examples
unzip archive.zip

This command extracts all files from archive.zip into the current directory.

unzip -d /path/to/extract archive.zip

This command extracts all files from archive.zip into the specified directory /path/to/extract.
