cd Command

The cd command in Unix/Linux is used to change the current working directory.

Syntax
cd [directory]
Options

The cd command does not have any options.

Description

The cd command is used to change the current working directory in Unix/Linux operating systems. It allows you to navigate between different directories.

Examples
cd /path/to/directory
cd ..
cd ~
