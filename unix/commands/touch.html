touch Command

The touch command in Unix/Linux updates the access and modification times of each file specified by the file(s) parameter. If the file doesn't exist, it creates an empty file with the specified name.

Syntax
touch [options] file(s)
Description

The touch command updates the access and modification times of file(s). If the file doesn't exist, it creates an empty file. This command is useful for creating placeholder files or updating timestamps.

Options
  • -a: Change only the access time.
  • -m: Change only the modification time.
  • -c: Do not create the file if it doesn't exist.
Examples
touch file1.txt

This command updates the access and modification times of file1.txt or creates an empty file if it doesn't exist.

touch -c newfile.txt

This command updates the timestamp of newfile.txt if it exists; otherwise, it does nothing.
