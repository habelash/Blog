ps Command

The ps command in Unix/Linux displays information about processes running on the system.

Syntax
ps [options]
Options
  • aux - Display all processes in a user-oriented format
  • e - Display all processes, including those without a controlling terminal
  • f - Display processes in a full-format listing
Examples
ps aux
ps -ef
ps -eLf
