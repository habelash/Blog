zip Command

The zip command in Unix/Linux compresses files into a ZIP archive.

Syntax
zip [options] zipfile.zip files...
Description

The zip command creates a ZIP archive containing specified files or directories. It compresses files using compression algorithms like deflate.

Options
  • -r: Recursively include all files in directories.
  • -m: Move the original files into the ZIP archive.
  • -q: Quiet mode (suppress output).
  • -v: Verbose mode (show more information during compression).
Examples
zip archive.zip file1.txt file2.txt

This command creates archive.zip containing file1.txt and file2.txt.

zip -r data.zip data_folder

This command recursively creates data.zip containing all files and folders within data_folder.
