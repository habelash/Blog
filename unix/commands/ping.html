ping Command

The ping command in Unix/Linux is used to test the reachability of a host on an IP network and measure the round-trip time for packets sent from the originating host to a destination computer.

Syntax
ping [options] hostname_or_ip_address
Options
  • -c count - Stop after sending count ECHO_REQUEST packets
  • -i interval - Wait interval seconds between sending each packet
  • -t timeout - Specify a timeout, in seconds, before ping exits regardless of how many packets have been received
Examples
ping example.com
ping -c 5 -i 1 example.com
