du Command

The du command in Unix/Linux displays disk usage statistics for files and directories.

Syntax
du [options] [file(s) or directory]
Description

The du command calculates and displays the disk usage of files and directories. It recursively summarizes disk usage of each directory and its subdirectories.

Options
  • -h: Print sizes in a human-readable format (e.g., 1K, 234M, 2G).
  • -s: Display only a total for each argument.
  • -c: Display a grand total.
Examples
du -h /var/log

This command displays the disk usage of the /var/log directory in a human-readable format.

du -s /home/user/docs

This command displays the total disk usage of the /home/user/docs directory.

du -h -c /home/user/docs /var/www

This command displays the disk usage of both /home/user/docs and /var/www directories with a grand total at the end.
