head Command

The head command in Unix/Linux displays the first part of a file or input (by default, the first 10 lines), useful for previewing file contents quickly.

Syntax
head [options] [file(s)]
Description

The head command outputs the first part of files. By default, it displays the first 10 lines, but can be customized to show more or fewer lines or bytes.

Options
  • -n number: Output the first number of lines.
  • -c bytes: Output the first bytes of the file.
Examples
head file.txt

This command displays the first 10 lines of file.txt.

head -n 5 data.csv

This command displays the first 5 lines of data.csv.

head -c 1024 largefile.bin

This command displays the first 1024 bytes of largefile.bin.
