df Command

The df command in Unix/Linux displays disk space usage for file systems.

Syntax
df [options] [file(s) or directory]
Description

The df command shows the amount of disk space used and available on file systems. It reports file system disk space usage including total, used, and available space.

Options
  • -h: Print sizes in a human-readable format (e.g., 1K, 234M, 2G).
  • -T: Display file system type.
  • -i: Display inode information instead of block usage.
Examples
df -h

This command displays disk space usage in a human-readable format.

df -T /home

This command displays the file system type for the /home directory.

df -i

This command displays inode information instead of block usage.
