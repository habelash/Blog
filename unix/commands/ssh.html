ssh Command

The ssh command in Unix/Linux is used to securely connect to a remote server or system.

Syntax
ssh [options] [user@]hostname [command]
Options
  • -p port - Connect to the specified port on the remote host
  • -i identity_file - Use the specified private key file for authentication
  • -l login_name - Specify the login name on the remote system
Examples
ssh username@example.com
ssh -p 2222 user@example.com
