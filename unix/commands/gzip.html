gzip Command

The gzip command in Unix/Linux compresses files using the gzip compression algorithm.

Syntax
gzip [options] file1 file2 ...
Description

The gzip command compresses files using the gzip compression algorithm, replacing each file with a compressed version having the extension .gz.

Options
  • -d: Decompress (unzip) compressed files.
  • -f: Force compression or decompression, even if it would overwrite existing files.
  • -k: Keep the original file(s) (don't delete it after compression or decompression).
  • -v: Verbose mode (show more information during compression or decompression).
Examples
gzip file1.txt

This command compresses file1.txt into file1.txt.gz.

gzip -d file1.txt.gz

This command decompresses file1.txt.gz back into file1.txt.
