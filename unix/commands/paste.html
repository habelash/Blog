paste Command

The paste command in Unix/Linux merges lines of files. It concatenates corresponding lines of the given input files, separated by a specified delimiter, and outputs them to standard output.

Syntax
paste [options] file1 file2...
Description

The paste command concatenates corresponding lines from multiple files or standard input and outputs them, separated by tabs by default or as specified by options.

Options
  • -d delimiter: Use delimiter to separate merged lines instead of tabs.
  • -s: Concatenate all lines of each file in parallel, rather than in sequence.
  • -z: Combine lines from different files sequentially into a single line, separated by null bytes.
Examples
paste file1.txt file2.txt

This command merges corresponding lines from file1.txt and file2.txt separated by tabs.

paste -d "," file1.txt file2.txt

This command merges lines from file1.txt and file2.txt, separated by commas instead of tabs.
