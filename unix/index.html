Unix Commands

File System Commands

Unix-like operating systems provide a vast array of commands for various purposes. Here's a categorized list of some common Unix commands:

  • ls - List directory contents
  • cd - Change directory
  • pwd - Print working directory
  • mkdir - Make directories
  • rmdir - Remove empty directories
  • cp - Copy files and directories
  • mv - Move (rename) files
  • rm - Remove files or directories
  • touch - Change file timestamps or create empty files
  • chmod - Change file modes or Access Control Lists
  • chown - Change file owner and group
  • ln - Create links between files

Text Processing Commands

  • cat - Concatenate and display files
  • head - Output the first part of files
  • tail - Output the last part of files
  • less - View file contents interactively
  • grep - Search files for lines matching a pattern
  • sed - Stream editor for filtering and transforming text
  • awk - Pattern scanning and processing language
  • sort - Sort lines of text files
  • uniq - Report or omit repeated lines
  • cut - Remove sections from each line of files
  • paste - Merge lines of files
  • tr - Translate or delete characters

System Information Commands

  • uname - Print system information
  • uptime - Show how long the system has been running
  • who - Show who is logged on
  • w - Show who is logged on and what they are doing
  • top - Display Linux tasks
  • ps - Report a snapshot of current processes
  • df - Report file system disk space usage
  • du - Estimate file space usage
  • free - Display amount of free and used memory in the system

Networking Commands

  • ping - Send ICMP ECHO_REQUEST to network hosts
  • ifconfig - Configure a network interface
  • netstat - Print network connections, routing tables, interface statistics, masquerade connections, and multicast memberships
  • traceroute - Print the route packets take to network host
  • ssh - OpenSSH SSH client (remote login program)
  • scp - Secure copy (remote file copy program)
  • ftp - File Transfer Protocol client
  • wget - Non-interactive network downloader
  • curl - Transfer data from or to a server

Package Management Commands

  • apt-get - APT package handling utility (Debian-based systems)
  • yum - Package manager for RPM-based distributions (e.g., CentOS, Fedora)
  • dnf - Package manager for RPM-based distributions (modern replacement for yum)
  • brew - Package manager for macOS (Homebrew)

File Manipulation Commands

  • tar - Tape archiver
  • gzip - Compress files
  • gunzip - Decompress files
  • zip - Package and compress files
  • unzip - Unpackage and extract files from ZIP archives

Miscellaneous Commands

  • date - Print or set the system date and time
  • cal - Display a calendar
  • echo - Display a line of text
  • history - Command history
  • kill - Send a signal to a process
  • shutdown - Shutdown or restart the system
  • reboot - Reboot the system
