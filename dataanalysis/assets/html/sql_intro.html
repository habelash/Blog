Introduction to SQL

Discover the fundamentals of SQL, a powerful language for managing and querying relational databases.

1. Introduction

SQL (Structured Query Language) is the standard language used to communicate with relational databases. It is essential for querying, updating, and managing data within a relational database system. SQL allows users to perform various operations such as data retrieval, insertion, updating, and deletion with a simple and intuitive syntax.

2. Key Concepts

Data Definition Language (DDL)

  • CREATE TABLE: Defines a new table along with its columns and data types.
  • ALTER TABLE: Modifies an existing table's structure, such as adding or removing columns.
  • DROP TABLE: Deletes an existing table and its data.

Data Manipulation Language (DML)

  • SELECT: Retrieves data from one or more tables based on specified criteria.
  • INSERT INTO: Adds new rows to a table.
  • UPDATE: Modifies existing data within a table.
  • DELETE: Removes rows from a table based on specified conditions.

Data Querying

  • WHERE: Filters records based on specified conditions.
  • JOIN: Combines rows from two or more tables based on a related column.
  • GROUP BY: Groups rows sharing a property and performs aggregate functions.
  • ORDER BY: Sorts the result set based on one or more columns.

3. Common Queries

  • Basic Select Query: SELECT * FROM table_name;
  • Filtered Query: SELECT * FROM table_name WHERE condition;
  • Join Query: SELECT columns FROM table1 JOIN table2 ON table1.id = table2.id;
  • Grouped Query: SELECT column, COUNT(*) FROM table_name GROUP BY column;

4. Example Code

-- Create a table
CREATE TABLE Employees (
    ID INT PRIMARY KEY,
    Name VARCHAR(100),
    Age INT,
    Department VARCHAR(50)
);

-- Insert data into the table
INSERT INTO Employees (ID, Name, Age, Department)
VALUES (1, 'Alice', 30, 'HR'),
       (2, 'Bob', 25, 'IT'),
       (3, 'Charlie', 35, 'Finance');

-- Select all data from the table
SELECT * FROM Employees;

-- Update a record
UPDATE Employees
SET Age = 31
WHERE Name = 'Alice';

-- Delete a record
DELETE FROM Employees
WHERE ID = 2;
